Containerize your platform

Containerization is a lightweight virtualization of a VM. In a container you run your application, and all features of an operating system are available.

The biggest difference between traditional hypervisors and container server, is that containers share linux kernel used by the operating system running the host machine, the means that consume of resources for all containers are through the same linux kernel:
